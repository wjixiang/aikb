--------------------------------------------------------------------------------
API ENDPOINTS
--------------------------------------------------------------------------------

Conversion:
  POST /api/v1/conversion/upload
    - Upload PDF and convert
  
  POST /api/v1/conversion/convert
    - Convert uploaded file
  
  GET /api/v1/conversion/formats
    - List supported formats

Chunking:
  POST /api/v1/chunking/chunk
    - Chunk text (default strategy)
  
  POST /api/v1/chunking/chunk/fixed
    - Fixed-size chunking
  
  POST /api/v1/chunking/chunk/semantic
    - Semantic chunking

Health:
  GET /health
    - Health check
  
  GET /health/ready
    - Readiness check
  
  GET /health/live
    - Liveness check

--------------------------------------------------------------------------------
QUICK START
--------------------------------------------------------------------------------

1. Install dependencies:
   cd apps/file-renderer
   pip install uv
   uv sync

2. Configure:
   cp .env.example .env
   # Edit .env as needed

3. Run:
   ./scripts/start.sh
   # or
   uv run python -m uvicorn main:app --reload --port 8001

4. Test:
   curl http://localhost:8001/health
   # Visit http://localhost:8001/docs for API documentation

--------------------------------------------------------------------------------
USAGE EXAMPLES
--------------------------------------------------------------------------------

1. Upload and convert PDF:
   curl -X POST "http://localhost:8001/api/v1/conversion/upload" \
     -F "file=@document.pdf" \
     -F "output_format=markdown"

2. Chunk text:
   curl -X POST "http://localhost:8001/api/v1/chunking/chunk" \
     -H "Content-Type: application/json" \
     -d '{"text": "...", "chunking_strategy": "semantic"}'
