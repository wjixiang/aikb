# 一百八十四、尿路结石的表现及并发症

# 1. 表现：

① 上尿路结石。

1)临床表现：疼痛（腰痛）+血尿。  
2）肾结石：肾区叩痛，肋脊角叩痛

② 下尿路结石

1)临床表现：小儿男孩+排尿突然中断+疼痛放射至远端尿道/阴茎头部+伴膀胱刺激征（上尿路无）+改变体位后继续排尿。

# 一百七十九、尿三杯的意义

1.全程血尿：①上尿路（肾和输尿管）病变（少数肾结核可为全程肉眼血尿）
