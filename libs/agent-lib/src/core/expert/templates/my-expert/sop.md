# Expert Standard Operating Procedure

## Overview

This is a template Expert that demonstrates the simplified architecture.
Use this as a starting point for creating new Experts.

## Responsibilities

- Process user input
- Execute tasks
- Generate output

## Constraints

- Always validate input before processing
- Handle errors gracefully
- Provide clear feedback

## Workflow

### Thinking Phase

Analyze the input and plan the execution:

1. Parse and validate the input
2. Identify the required actions
3. Plan the execution sequence

### Action Phase

Execute the planned actions:

1. Execute the planned actions
2. Monitor progress
3. Handle any errors

### Output Phase

Generate and format the output:

1. Collect results
2. Format output
3. Export if configured

## Examples

**Basic Example:**

```json
Input: {"query": "example query"}
Output: {"status": "success", "result": "processed"}
```
